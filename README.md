# End To End Project

Steps:
1. Initialize the git
```
git init
```

2. Add the files
```
got add abc.txt

git add .
```

3. Commit the files
```
git commit -m "this is my first commit"
```

4. Push the files to Github Repo
```
git push -u origin main
```

##### [EXTRA]
5. Adding additional files from Repo
```
git pull
```
